This PowerShell script sorts each cube then sorts the cubes, which computed the second line of each entry below:
$cubes = type .\sortme.txt | %{ $_ -split " " } | %{ $_.ToCharArray() | sort | Out-String -NoNewline } | sort; $cubes -join " " | Out-File .\sorted.txt
